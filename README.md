## Project Description
This project presents a structured HTML-based recreation of the historical Pakistani newspaper *The Nawa-i-Pakistan*. The objective of the project was to visually simulate a traditional newspaper layout by employing multi-column formatting, typographic hierarchy, and structured content placement using HTML and CSS.

## Features Implemented
- Newspaper-style multi-column layout using CSS Flexbox  
- Prominent masthead and headline hierarchy to reflect print journalism  
- Use of serif typography to emulate classic newspaper aesthetics  
- Structured placement of articles with borders and separators  
- Inline CSS styling to achieve precise layout control  

## Challenges Faced
A key challenge in this project was maintaining consistent column alignment while preserving the visual authenticity of a traditional newspaper format. This issue was addressed through careful use of flex containers, spacing adjustments, and border styling.
